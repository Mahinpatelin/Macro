This repository contains my custom macropad project, built specifically for Hack Club to help streamline my everyday tasks and boost my productivity. I designed the macropad from scratch using KiCad for the custom PCB, which is powered by a Seeed Studio XIAO RP2040 microcontroller. The board features 8 fully programmable mechanical keys for everyday shortcuts and a rotary encoder currently set up for volume control and a push-to-mute function. It also sits perfectly inside a custom 3D-printed enclosure that I generated using OpenSCAD and TurboCase. Inside this repository, you will find everything you need to build or modify it yourself, including the 3D case files, the PCB schematics, and the data-driven QMK firmware source code. If you want to flash the firmware yourself, simply drop the provided firmware folder into your local QMK setup, compile it using the QMK CLI, and drag the generated uf2 file onto the RP2040 while it is in bootloader mode.
